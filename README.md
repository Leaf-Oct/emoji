# Emoji

一个emoji的大集合, 以最小最精简的方式解决国内输入法的emoji普遍存在的问题: Emoji不全

比方说, 「🥵🥵行秋, 嘿嘿🥵🥵我的行秋🥵」, 🥵这个表情好像见不到(至少在讯飞输入法里见不到)

设计灵感来源于Ubuntu系统下自带的Character软件
